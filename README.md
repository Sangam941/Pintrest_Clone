# Pinterest Clone (MERN Stack)

A Pinterest-like web application built using **MongoDB, Express, React, Node.js** (MERN stack) with user authentication, image upload, and pin management. Users can register, login, create pins with images, follow other users, and view pins in real-time.

---

## Features

- User registration and login with JWT stored in cookies  
- Create, view, and delete pins  
- Upload images to Cloudinary  
- Follow and unfollow users  
- View other usersâ€™ profiles  
- Fully responsive UI  
- Real-time updates for pins (optional if you integrate sockets)

---

## Tech Stack

- **Frontend:** React, Axios, React Router, Tailwind CSS  
- **Backend:** Node.js, Express.js, MongoDB (Mongoose)  
- **Authentication:** JWT with cookies  
- **Image Storage:** Cloudinary  
- **Other:** react-hot-toast for notifications, multer for file uploads
